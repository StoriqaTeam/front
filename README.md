**Пререквизиты:**
1. склонируйте репозиторий
2. перейдите в созданную папку
3. установите зависимости: `yarn`

---

**Запуск версии для разработки:**

`yarn start`

*http://localhost:3000/*

---

**Server-side-rendering версия:**

`yarn build && yarn server`

*http://localhost:3003/*

---

**Продакшн-билд:**

`yarn build`
